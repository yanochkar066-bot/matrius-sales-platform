# Matrius Sales Platform

Внутренняя платформа для менеджеров по продажам онлайн-школы [Matrius](https://matrius.online). Одностраничное веб-приложение с картой курсов, инструментами для работы с клиентами и материалами для отдела продаж.

---

## Проблема

Менеджер по продажам онлайн-школы работает с 8 курсами одновременно. В процессе звонка нужно быстро ориентироваться в ценах, возрастах, форматах и уметь отвечать на возражения — без переключения между таблицами, папками и чатами.

До создания этой платформы информация была разбросана по Google-таблицам, PDF-файлам и мессенджерам: менеджер тратил время на поиск нужных данных прямо во время разговора с клиентом, допускал ошибки в ценах и не всегда мог быстро отправить нужную ссылку родителю.

**Решение:** единое рабочее место, где вся информация структурирована и доступна в 1–2 клика — карта курсов, цены, скрипт, ссылки для родителей и генератор прайс-карточек.

---

## Возможности

### Карта курсов
- Все 8 направлений школы на одном экране: ментальная арифметика, скорочтение, математика, логопедия, подготовка к школе, русский язык, английский язык, IT для детей
- Фильтрация по возрасту
- Поиск по названию и описанию
- Детальная карточка курса: описание, результаты, форматы, цены, возражения с готовыми ответами
- Сравнение двух курсов

### Все цены
- Таблица абонементов по каждому курсу: 3, 6, 9, 12 месяцев
- Индивидуальный и групповой форматы

### Генератор прайс-карточек
- Конструктор изображений для отправки клиентам
- Настраиваемые поля: название, период, количество уроков, цена за урок, цена абонемента, цена в месяц
- Экспорт в PNG через html2canvas

### Презентации
- Быстрый доступ к Google Slides по каждому курсу
- Копирование ссылки в один клик для отправки родителям

### Родителям
- Ссылки на сайт, страницу записи, отзывы, лицензию, Telegram-канал, ВКонтакте
- Копирование ссылки в буфер одним нажатием

### Скрипт продаж
- Пошаговый скрипт разговора с клиентом
- Готовые фразы и ответы на возражения по этапам

## Технологии

- Vanilla JS — без фреймворков
- HTML5 + CSS3 (Flexbox, Grid, CSS-переменные)
- [Onest](https://fonts.google.com/specimen/Onest) — шрифт бренда (Google Fonts)
- [html2canvas](https://html2canvas.hertzen.com/) — экспорт карточек в PNG
- `navigator.clipboard` — копирование ссылок
- `localStorage` — хранение данных
- Адаптивная верстка: десктоп, планшет, мобильный (от 320px)

## Запуск

Проект не требует сборки — достаточно открыть `courses-map.html` в браузере или запустить локальный сервер:

```bash
python3 -m http.server 9002
```

Затем открыть [http://localhost:9002/courses-map.html](http://localhost:9002/courses-map.html)
