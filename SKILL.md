# MATRIUS — Brand Skill

## О бренде

**MATRIUS** — онлайн-школа будущего для развития детей с гарантией результата.  
Работает 7+ лет, 50 000+ учеников, резидент «Сколково», имеет образовательную лицензию.

**Главный слоган:** «Школа будущего для развития детей с гарантией результата!»

**Суть позиционирования:**  
Мы учим иначе, потому что знаем, как работает мозг. Научный подход к развитию ребёнка + человеческий язык + практическая польза для семьи.

**Аудитория:** родители детей 4–17 лет, которые хотят развить у ребёнка самостоятельность, интерес к обучению и навыки будущего — без давления и стресса.

## Ценности и философия бренда

1. **Знаем, как работает мозг** — методология основана на нейронауке и детской психологии

## Голос бренда (Tone of Voice)

**Роль:** заботливый эксперт — как умный друг с психологическим образованием, который говорит честно и по делу.

**К родителям:** на **«вы»**, тепло, уважительно, без менторства.  
**К детям:** на **«ты»**, дружелюбно, с юмором и энергией.

**Как писать:**
- Конкретика и польза вместо абстракций: не «уникальная методика», а «первые результаты через месяц»
- Коротко и ёмко: один абзац — одна мысль
- Факты и цифры: 50 000 учеников, 7 лет, 700 слов/мин
- Без канцелярита: «поможем разобраться» вместо «осуществляем поддержку»
- Без давления: не «срочно запишитесь», а «попробуйте бесплатный урок»

**Запрещённые слова и обороты:**
- «Уникальный», «революционный», «лучший в мире»
- «Гарантируем успех» (только через конкретику: «успеваемость в плюс через 3 месяца»)
- Канцелярит и сложные термины без объяснения

## Цветовая система

```
Тёмно-синий (navy):   #1C3152   ← обложки, хедеры, тёмные акцентные плашки
Синий (карточки):     #2D5187   ← основной фон синих карточек (с градиентом до #3A6494)
Оранжевый:            #EE6435   ← главный акцент, нумерация, CTA, подчёркивания
Фон страницы:         #F7F8FA   ← внутренние страницы гайдов (не чистый белый)
Белый:                #FFFFFF   ← светлые карточки, хедер страницы
Текст заголовка:      #1A2B42   ← заголовки на светлом фоне
Текст тела:           #4A5568   ← основной текст в карточках и блоках
Граница:              #E2E6EE   ← обводка светлых карточек, разделители
Оранжевый мягкий:     #FFF5F1   ← фон tip-карточек
Оранжевый тёмный:     #CC4A1A   ← текст внутри tip-карточек
```

**Правила сочетания:**
- Тёмный фон (обложки, финальные страницы): `#1C3152` + белый текст + оранжевый акцент
- Светлый фон (гайды, внутренние страницы): `#F7F8FA` + хедер `#FFFFFF` + синие карточки
- CTA-кнопки: оранжевый `#EE6435` фон, белый текст

## Типографика

**Шрифты:**
```
Onest Bold (700)    — обложки, H1/H2, заголовки карточек, бейджи, нумерация
Onest SemiBold (600)— подзаголовки разделов, акцентные строки
Onest Medium (500)  — метки, навигация, caption, второстепенные заголовки
Onest Regular (400) — основной текст, описания, списки
Caveat (курсив)     — декоративный акцент: 1 раз на странице, не больше
```

**Размеры для PDF (A4 = 794px):**
```
Обложка H1:          80–120px  Bold, letter-spacing -0.04em
Заголовок страницы:  24–32px   Bold
Вводный текст:       22–28px   SemiBold, letter-spacing -0.03em
Название карточки:   14–17px   Bold
Текст карточки:      12.5–13.5px  Regular, line-height 1.6–1.65
Пример/цитата:       12–13px   Regular, line-height 1.7
Метка/тег:           10–12px   Bold, uppercase, letter-spacing 1.5px
Caveat-акцент:       22–28px   курсив
```

**Размеры для веб:**
```
H1 hero:       56–80px
H2 секция:     36–48px
H3 карточка:   22–28px
Body:          16–18px, line-height 1.6
Caption:       13–14px
```

## Визуальный стиль

### Система карточек (3 типа — только они)

**1. Синяя карточка** (`card-blue`) — основные смысловые блоки, нумерованные шаги, привычки, правила.
- Фон: градиент `#2D5187 → #3A6494` (135deg)
- Оранжевый круглый бейдж с цифрой/символом
- Белый текст заголовка + `rgba(255,255,255,0.82)` для описания
- Верхний блик: `::before` 1px `rgba(255,255,255,0.18)`

**2. Светлая карточка** (`card-light`) — дополнительная информация, сравнения, примеры.
- Фон `#FFFFFF`, граница `1.5px solid #E2E6EE`, border-radius 14–16px

**3. Подсказка** (`card-tip`) — важный момент, предупреждение, лайфхак (max 1–2 на странице).
- Фон `#FFF5F1`, левая полоса `3px solid #EE6435`
- Текст `#CC4A1A` или `#4A5568`

### Обложка
- Фон `#1C3152` + сетка (`linear-gradient` 30px, `rgba(255,255,255,0.04)`)
- Типографика: крупный Onest Bold белый + оранжевый акцент на ключевом слове
- Логотип (кружок) — верхний левый угол
- Метка «5+» или возраст — оранжевая, верхний правый угол (только обложки)
- Фото/персонаж — опционально, снизу, `mix-blend-mode: screen` для фото с тёмным фоном

### Общие правила вёрстки
- Хедер страницы: белый фон + логотип слева + «Полезные привычки» и номер страницы справа
- Разделитель под хедером: `1px solid #E2E6EE`, `margin: 0 44px`
- Контентная область: фон `#F7F8FA`, padding `22px 44px 34px`
- gap между карточками: 8–20px (PDF плотнее, чем веб)
- border-radius карточек: 12–18px
- Горизонтальные отступы страницы: 44–48px

## Логотип

- **Круглый** (`logo-white-bg.png`) — хедер внутренних страниц, светлый фон
- **На тёмном фоне** (`logo-dark-bg.png`) — обложка, финальная страница, тёмный фон
- **Размер в хедере:** 36–44px
- **Размер на обложке:** 68–80px
- Запрещено: менять цвета, растягивать, применять `filter: brightness()`

```
Стр. 1:  Обложка — тёмно-синяя, крупный заголовок, логотип, подзаголовок
```

## Антипаттерны — не делать

**Дизайн:**
- ❌ Чистый белый `#FFFFFF` как фон страницы — используй `#F7F8FA`
- ❌ Flat-цвет для синих карточек — только градиент `#2D5187 → #3A6494`
- ❌ Тёмный фон `#385681` — слишком светлый, используй `#1C3152`
- ❌ `box-shadow` на карточках в PDF — не рендерится, используй `border`
- ❌ border-radius > 20px на мелких элементах — выглядит «пузырём»
- ❌ Caveat на каждом абзаце — максимум 1 раз на странице
- ❌ Оранжевый для длинных текстов — только метки, заголовки, бейджи
- ❌ 3D-иллюстрации без реального файла — не вставлять placeholder

**Тексты:**
- ❌ «Уникальный», «лучший», «революционный» — клише без доказательств
- ❌ Менторский тон и поучения — только советы и факты
- ❌ Более 3 восклицательных знаков на странице — снижает доверие
- ❌ Сложные термины без объяснения — аудитория не обязана знать нейронаучный жаргон
