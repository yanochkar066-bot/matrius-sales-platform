# Matrius Sales Platform

Платформа менеджера по продажам онлайн-школы [Matrius](https://matrius.online). Одностраничное веб-приложение с картой курсов, инструментами для работы с клиентами и материалами для отдела продаж.

---

## Проблема

Менеджер онлайн-школы ведёт переговоры по 8 курсам одновременно. Во время звонка нужно мгновенно находить цены, возрасты, форматы, отвечать на возражения и отправлять материалы родителю — без пауз и ошибок. До платформы вся информация была разбросана по Google-таблицам, PDF и чатам: менеджер переключался между вкладками, называл устаревшие цены и терял время на поиск нужной ссылки прямо в разговоре.

---

## Пользователь

Менеджеры по продажам онлайн-школы Matrius. Работают с входящими заявками, ведут переговоры с родителями, консультируют по курсам и закрывают сделки.

---

## Основная функция

**Вход:** менеджер открывает платформу перед звонком или в процессе.

**Что делает:**
- Показывает все 8 курсов с фильтрацией по возрасту и поиском
- Раскрывает детали курса: описание, результаты, форматы, цены, готовые ответы на возражения
- Даёт единую таблицу всех цен по абонементам (3/6/9/12 месяцев)
- Генерирует прайс-карточку в PNG для отправки в мессенджер
- Открывает и копирует ссылки на презентации по каждому курсу
- Копирует нужные ссылки для родителя (сайт, запись, отзывы, лицензия, соцсети)
- Ведёт по скрипту продаж с фразами по этапам

**Выход:** готовая карточка с ценой, ссылка для клиента, ответ на возражение — в 1–2 клика.

---

## Формат результата

- PNG-изображение прайс-карточки (экспорт через html2canvas)
- Ссылка в буфере обмена — для вставки в мессенджер
- Интерактивный интерфейс — без скачивания, работает в браузере

---

## Минимальный функционал (MVP)

Реализовано в первой версии:
- Карта курсов с фильтрами и детальными карточками
- Сравнение двух курсов
- Таблица всех цен
- Генератор прайс-карточек с экспортом в PNG
- Модалка «Родителям» с копированием ссылок
- Модалка «Презентации» с быстрым доступом к Google Slides
- Скрипт продаж по этапам
- Адаптивная версия для мобильного

Планируется:
- Личный кабинет менеджера с авторизацией
- Заполнение цен через интерфейс (без правки кода)
- Статистика обращений по курсам

---

## Технологии

- Vanilla JavaScript — логика, фильтрация, модалки, генерация карточек
- HTML5 + CSS3 — Flexbox, Grid, CSS-переменные, адаптив от 320px
- [html2canvas](https://html2canvas.hertzen.com/) — экспорт прайс-карточки в PNG
- `navigator.clipboard` API — копирование ссылок
- [Onest](https://fonts.google.com/specimen/Onest) (Google Fonts) — фирменный шрифт бренда
- Без фреймворков и сборщиков — запускается открытием одного HTML-файла

## Запуск

Проект не требует сборки — открыть `courses-map.html` в браузере или запустить локальный сервер:

```bash
python3 -m http.server 9002
```

Затем открыть [http://localhost:9002/courses-map.html](http://localhost:9002/courses-map.html)
